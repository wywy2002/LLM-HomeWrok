# Task Board Node

## Stack

- Backend: Node.js built-in `http` server
- Frontend: Vanilla HTML/JS
- Persistence: JSON file

## Run

From this directory:

```powershell
npm start
```

Then open `http://127.0.0.1:3001`.

## Features

- Create tasks
- List tasks
- Toggle completion
- Delete tasks

## Notes

- Data is stored in `data/tasks.json`.
- This version avoids external npm dependencies to keep the project lightweight.
