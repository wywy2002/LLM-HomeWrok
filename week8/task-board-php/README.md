# Task Board PHP

## Stack

- Backend: PHP
- Frontend: Vanilla HTML/JS
- Persistence: SQLite

## Run

From this directory:

```powershell
php -S 127.0.0.1:3002
```

Then open `http://127.0.0.1:3002`.

## Features

- Create tasks
- List tasks
- Toggle completion
- Delete tasks

## Notes

- Data is stored in `tasks.sqlite`.
- This is the non-JavaScript stack version required by the assignment.
